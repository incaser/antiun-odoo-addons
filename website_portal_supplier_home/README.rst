============================
Website Portal Supplier Home
============================

This module was written to extends the functionality of website_portal module
to support render a custom home page layout.

Installation
============

To install this module, you need to:

* Only install the module in *Configuration > Local Modules*

Configuration
=============

To configure this module, you need to:

* Nothing to do.

Usage
=====

To use this module, you need to:

* go to login and click to *My Account* into top menu.
